## Overview 

Dt Money is a web app to control finances.  
Developed with React and Typescript.

Used Hooks:   
- useState
- useEffect
- useContext
- useReducer
- useCallback
- useMemo

Used Packages:  
- axios 
- use-context-selector  

It was developed as part of the Rocketseat Full-stack MBA. (https://www.rocketseat.com.br/mba)
